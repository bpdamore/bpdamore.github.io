24 Week Data Anlytics Boot Camp

Since February I have been participating in a Data Analytics Boot Camp at Vanderbilt University. The goal of taking this course was to be able to make a shift in my career from hospitality and coffee production into a data based role. While working 40 hours a week at Stay Golden, I was also attending 10 hours of classes a week and putting 20 hours of outside work into homework and projects.

    • Python

    Prior to this class I had a rudimentary knowledge of Python, learning solely through Google and Stack Overflow. The main bulk of this program focused on using Python and various modules inside of Python to manipulate and visualize data.
    We were encouraged to mix and match these concepts to accomplish our goals.

  • Utilizing pandas, we were able to programmatically manipulate and analyze CSV files and tables found on the internet.
  • Splinter and Beautiful Soup allowed us to use the internet as our dataset. Anything interesting found on the internet can be scraped and analyzed through Beautiful soup and Python.
  • Matplotlib was used to create basic visualizations of data with Python.
  • SQLAlchemy and Pymongo allowed us to connect to SQL and NoSQL databases in Python, and to use those datasets.


    • Database Management

    Using SQL and MongoDB, we went over the basics of connecting to, and utilizing data. We covered basics of PostgreSQL, SQLite, and MySQL.

  • Using Amazon Web Services, we created a PostgreSQL database, and connected it on multiple machines.
  • We covered the ETL (Extract, Transform, Load) process, allowing us to manipulate and add to current databases.


    • Web Development

    Later portions of the class dealt with basic Web Development in order to help us present our findings or visualizations.
    This website is built from scratch, using concepts we went over.

  • HTML and CSS helped us to create the structure and look of the website
  • Javascript showed us how to create interactive graphs and events
  • Plotly was used to easily create interactive graphs
  • D3 was used to create fully custom visualizations that were interactive.
